build/src/common/property-table.component.ts:213

public getDisplayValue(key): Any

Returns the display value for the given simpleKeys key.

The display value is determined as follows:

  • If valueChoices is set and the key/value combination is the choices lookup, then that lookup target is the display value.
  • Otherwise, if the format function is set, then this method delegates to that function.
  • Otherwise, if the value is boolean, then the display value capitalizes the boolean string value, e.g. True.
  • Otherwise, the string conversion of the value itself is displayed.
