build/src/controls/player.component.ts:78

public ngOnChanges( )

Handles a value change or cancel request.

The state machine is as follows:

next|previous -> clear play; request -> receive ->
clear waiting flag

play -> set play; request -> receive -> repeat

If the isWaiting flag is set, then the request is queued. Otherwise, the request is issued and the waiting flag is set.

When the request is fulfilled, if there is a queued request then fire the queued request and retain the waiting state. Otherwise, clear the waiting flag.

The repeat is paused, if necessary, until the delay time elapses.
