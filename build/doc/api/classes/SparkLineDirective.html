SparkLineDirective

Draws a D3 spark line which plots a uniform sequential one-based index vs values. The X values are the discrete one-based data indexes. The data items are the Y axis data points.

public domain: Any

The optional domain arrays, used as folows:

  • If the input is an array of two items, then the Y domain is that input.
  • Otherwise, if the input is a {x, y} object, e.g. {x: [0, 12], y: [1, 100]}, then the X and Y domains are the respective property values.

The default X or Y domain is the respective mapped data min and max.
