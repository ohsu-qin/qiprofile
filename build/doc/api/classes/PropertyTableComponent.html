The property name => label function, used as described in getLabel.

The return value can be unescaped HTML, i.e. embedded HTML is interpreted. For example, &tau;<sub>i</sub> displays as τi. A simple string return value is displayed as the string value. A null or undefined return is displayed as the getLabel default.
